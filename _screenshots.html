AI 회계 클로징 관련 - 기존 UI 현황 7개

1. 통장 AI 분류 UI 완성 / AI 엔진 없음

파일 업로드 + 컬럼 매핑 + AI 분류 결과 테이블 (신뢰도 프로그레스바, 계정과목 수정 드롭다운). 현재 더미 데이터 10건만 표시.

2. 결산준비 체크리스트 프레임워크 완성

D-day 카운트다운 + 매출/매입/비용 요약 + 자동체크 5개 + 수동체크 5개 + 세무사 자료 패키지 ZIP 다운로드. AI 분류 결과가 여기로 연결됨.

3. 세금계산서 완전 작동

매출/매입 탭 + 거래처별 집계 + 공급가액/세액 합계 + 부가세 예상 납부액. DB 연동 완료.

4. 카드 지출내역 완전 작동

카드별/부서별 필터 + CRUD + 사용구분(법인/개인) + 사업명/문서번호 연동. 10건 샘플 데이터.

5. 법인카드 정산 완전 작동

정산대기/완료 탭 + 규정준수 뱃지 + 일괄정산 + 승인내역. 결재 승인 시 자동정산 연동.

6. 서류 요청/알림 워크플로우 완성

세무사 ↔ 회사 간 서류 요청/업로드/확인 워크플로우. 상태 추적 + 파일 첨부 + 알림.

7. 세무 리포트 완전 작동

매출/매입 합계 + 부가세 예상 납부액 + 월별 차트 + 계정과목별 도넛차트 + 비중 테이블.

빠진 핵심: AI 분류 엔진 (LLM 연동) + 증빙 교차 매칭 (카드↔세금계산서↔통장) + 현금흐름 예측
